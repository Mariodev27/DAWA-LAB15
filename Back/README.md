# Mi-Proyecto

Este es un proyecto básico de Node.js con Express y Nodemon. La estructura del proyecto es la siguiente:

- `config`: Esta carpeta contiene archivos de configuración.
- `controllers`: Esta carpeta contiene los controladores de la aplicación.
- `models`: Esta carpeta contiene los modelos de la aplicación.
- `routes`: Esta carpeta contiene las rutas de la aplicación.
- `app.js`: Este es el archivo principal de la aplicación.

## Cómo ejecutar el proyecto

1. Asegúrate de tener Node.js y npm instalados en tu máquina.
2. Clona este repositorio en tu máquina local.
3. Navega a la carpeta del proyecto en la línea de comandos.
4. Ejecuta `npm install` para instalar las dependencias del proyecto.
5. Ejecuta `npm start` para iniciar la aplicación.

La aplicación ahora debería estar ejecutándose en `http://localhost:3000`.
